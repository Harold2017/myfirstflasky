    API

    1. Find your api token on Profile page.

    2. Add one sensor on Edit Sensors page.

       You will find your sensor id in the sensors table.
       Combine this sensor id and your Api Token, make a HTTP post as the following shows:

    http://OurWebSite/api/v2.0/Api_Token/Sensor_Id

    Post body: {"timestamp":UTC_time (or "None"), "value": number}

       Then check your data point on Chart page.

    3. For gps, the HTTP post should be:

    http://OurWebSite/api/v1.0/gps/Api_Token

    Post body: {"timestamp":UTC_time (or "None"), "lat": number, "lng": number}

       Then check your data point on GPS page.
